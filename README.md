# React Portfolio

A modern, responsive portfolio website built with React.js.

## Features

- **Responsive Design**: Mobile-first approach with responsive grid layouts
- **Smooth Navigation**: Sticky navbar with smooth scrolling
- **Project Showcase**: Display your best work with descriptions and technologies
- **Contact Form**: Functional contact form with form validation
- **Modern Styling**: Gradient backgrounds and smooth animations
- **Accessible**: Semantic HTML and ARIA labels

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open your browser and navigate to `http://localhost:3000`

## Available Scripts

- `npm start` - Runs the app in development mode
- `npm build` - Builds the app for production
- `npm test` - Runs the test suite

## Customization

### Update Personal Information

Edit the following components to add your own information:

- **Header.js**: Update name and tagline
- **About.js**: Add your bio and skills
- **Projects.js**: Update project details
- **Contact.js**: Add your contact information and social links

### Styling

All styles are in CSS files within the `src/components/` directory. The main color scheme uses:
- Primary: `#667eea`
- Secondary: `#764ba2`
- Background: `#f5f5f5`

Modify colors in the CSS files to match your brand.
